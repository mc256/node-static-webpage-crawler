# node-static-webpage-crawler

This is a very simple web crawler. You can use this to download your ENTIRE website.

## Install

```
npm install node-static-webpage-crawler
```

## Use it

```
node main.js --url=YOURDOMAIN --cache-dir=YOURDIRECTORY
```
